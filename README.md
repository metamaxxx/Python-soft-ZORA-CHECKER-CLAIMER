### 3. Настройка файлов

1. файл `wallets.txt` добавьте в него адреса кошельков (по одному на строку)

2. файл `private_keys.txt` добавьте в него приватные ключи (по одному на строку)
   - Важно: приватные ключи должны соответствовать адресам в том же порядке!

### 4. Запуск скрипта
1. Убедитесь, что виртуальное окружение активировано (в начале строки должно быть `(venv)`)
2. Запустите скрипт:
   ```
   python claim_checker.py
   ```
3. В меню выберите нужное действие:
   - 1 - Проверить аллокации всех кошельков
   - 2 - Проверить аллокацию одного кошелька
   - 3 - Клеймить токены для одного кошелька
   - 4 - Клеймить токены для всех кошельков
   - 5 - Выход

### 5. Проверка результатов
- После успешного клейма вы увидите хэш транзакции и ссылку на Basescan
- Можете проверить статус транзакции, перейдя по ссылке
- Для проверки успешности клейма используйте опцию 1 или 2 в меню

## Важные замечания
1. Не передавайте никому свои приватные ключи
2. Убедитесь, что у вас достаточно ETH в сети Base для оплаты газа
3. Если транзакция не проходит, проверьте:
   - Достаточно ли газа
   - Не был ли уже выполнен клейм
   - Есть ли аллокация на адресе

Если у вас возникли проблемы:
1. Проверьте, что все файлы настроены правильно
2. Убедитесь, что все зависимости установлены
3. Проверьте соответствие приватных ключей и адресов
